# Roadmap

The current roadmap lives in the repo as
[ROADMAP.md](https://github.com/Sigilweaver/OpenKSpace/blob/main/ROADMAP.md).

Headline near-term items:

- 0.3.x: refactor isolated `unsafe { mem::zeroed() }` in
  `openkspace-recon` and adopt workspace-wide
  `unsafe_code = "forbid"`.
- Non-Cartesian trajectory support (radial, spiral) behind a
  feature flag.
- Multi-band SMS (Wave-CAIPI) recon.

Suggestions welcome via
[GitHub issues](https://github.com/Sigilweaver/OpenKSpace/issues).
